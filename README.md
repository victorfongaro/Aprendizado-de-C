# 🎯 Jornada de Aprendizado: Lógica e Linguagem C

Este repositório foi criado para documentar e organizar toda a minha evolução nos fundamentos de Ciência da Computação, focado no desenvolvimento de lógica de programação e no domínio da **Linguagem C**. 

Aqui guardo os meus exercícios de fixação, desafios de algoritmos e projetos acadêmicos desenvolvidos ao longo do curso.

---

## 🗺️ Mapa de Conteúdo e Progresso

Aqui está a estrutura de tópicos que estou explorando, dividida por conceitos fundamentais de engenharia de software:

- [x] **01. Fundamentos e Sintaxe Básica**
  - Variáveis, tipos de dados (`int`, `float`, `char`) e modificadores.
  - Entrada e saída de dados (`printf`, `scanf`).
  - Operadores aritméticos e lógicos.
- [x] **02. Estruturas de Controle e Decisão**
  - Desvios condicionais (`if`, `else`, `switch-case`).
  - Operadores relacionais e tomada de decisão.
- [x] **03. Estruturas de Repetição (Loops)**
  - Laços de iteração (`while`, `do-while`, `for`).
  - Controle de fluxo (`break`, `continue`).
- [x] **04. Estruturas de Dados Homogêneas**
  - Vetores (Arrays unidimensionais).
  - Matrizes (Arrays multidimensionais e álgebra linear computacional).
- [x] **05. Modularização**
  - Criação de funções e protótipos.
  - Escopo de variáveis (locais e globais) e passagem de parâmetros.

*Legenda: `[x]` Concluído, `[/]` Em Andamento, `[ ]` Planejado.*

## 🧠 Filosofia de Desenvolvimento

Em Ciência da Computação, aprender C não é apenas aprender uma linguagem, mas sim entender **como o computador funciona por baixo dos panos**. O foco deste repositório é desenvolver:
1. **Gerenciamento manual de recursos:** Entender a fundo o uso de memória RAM.
2. **Otimização de Algoritmos:** Escrever códigos eficientes e de alta performance, essenciais para sistemas embarcados e engenharia pesada.
3. **Resolução de Problemas:** Quebrar problemas complexos em funções lógicas menores.
